# ❤️ Cardiovascular Disease Analysis & Prediction Application

Welcome to the **Cardiovascular Disease Analysis and Prediction** project! 🩺 This is a machine learning-powered web application built with **Streamlit** ✨ designed to analyze and predict the risk of cardiovascular disease based on various health metrics. It provides an interactive platform for exploratory data analysis (EDA) and real-time predictive insights.

---

## 🌟 Key Features

- **📊 Exploratory Data Analysis (EDA):** Dive deep into cardiovascular data with interactive visualizations to uncover trends and relationships between health indicators.
- **🧠 Advanced Machine Learning Model:** Leverages a pre-trained, robust ML model (`model.joblib`) to accurately predict the likelihood of cardiovascular disease.
- **💻 Interactive Web Interface:** A premium, user-friendly Streamlit web application that seamlessly collects health metrics and delivers immediate predictive feedback.
- **⚙️ Seamless Pipeline:** Well-structured data processing and modeling logic that mirrors the rigorous workflow from our foundational Jupyter Notebook.

---

## 📂 Project Structure

Here is a quick overview of what you'll find in this repository:

- 🚀 **`app.py`**: The main gateway to the Streamlit web application.
- ⚙️ **`model_logic.py`**: The core logic handling data preprocessing and machine learning predictions.
- 📉 **`eda_visuals.py`**: A dedicated suite of scripts for rendering beautiful data visualizations.
- 📦 **`requirements.txt`**: A complete list of Python package dependencies needed to run the app.
- 🤖 **`model.joblib`**: The heart of the prediction engine—our trained machine learning model.
- 📄 **`cardio_data_processed.csv`**: The meticulously cleaned dataset driving both analysis and modeling.
- 📓 **`cardiovascular-disease (1).ipynb`**: The original comprehensive Jupyter Notebook documenting exploratory data analysis and model training.

## 🛠️ How to Run Locally

Get the application up and running on your local machine in just a few simple steps!

1. **Let's navigate inside the project directory** (if you cloned it, or just use the existing one!).
   
2. **Install the dependencies** (we recommend using a virtual environment):
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch the web application**:
   ```bash
   streamlit run app.py
   ```
